This tool currently utilizes the PapaParse library in order to parse a JSON file back to CSV.
The JSON file root element must be an array. In the future this limitation will be removed.

[ 
  {  
    "Name": "Name1",
    "Value": "Value1"
  },
  {
    "Name": "Name2",
    "Value: "Value2"
  }
]
   
{{ csv }}
